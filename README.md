# Crazy_Bird
An interesting fun based game developed using Unity game engine.

Game Play: Player needs to survive by avoiding colliding with the auto-movable obstacles and should able to reach the goal point.

Rule: Player should be within a fixed game environment,otherwise he should lose his life. Only 3 lives should be permitted. After losing 3 lives the player should be redirected to the home page. The progression of player should be towards right.

Mechanics: Player should be able to move left, right and up. Minimum one obstacle is needed. Two levels should be provided. Variation in obstacles should be in 2nd level. Once player completes the game, Player won caption should be provided. Proper light effects which suits the game environment. Proper audio/music/sound effects which suits the game environment.
